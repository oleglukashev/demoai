# DemoAI — фронтенд (Next.js)

Страница чата с AI-ассистентом в дизайне table-flow.app. Работает в паре с
бекендом `../api` (NestJS + Prisma).

- **Слева** — sidebar: загрузка текстовых документов (drag&drop или клик).
  Файл уходит в API, где разбивается на чанки и сохраняется в БД; в списке
  показывается число чанков.
- **Снизу** — поле ввода сообщения (Enter — отправить, Shift+Enter — новая строка).
- **В теле** — появляется ваше сообщение, следом — ответ ассистента (RAG по
  загруженным документам) с указанием источников.

## Запуск

Сначала поднимите бекенд (`../api`, порт 3006), затем:

```bash
cd demoai/app
npm install
npm run dev      # http://localhost:3005
```

Адрес API задаётся в `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:3006
```

## Структура

- `app/page.tsx` — экран чата (сообщения + список документов из API)
- `app/components/Sidebar.tsx` — sidebar с загрузкой документов в API
- `app/globals.css` — дизайн-токены и стили TableFlow

Вся логика загрузки/чанкинга/ответов — в бекенде `../api`.
